# Песочница для криптокодов
Консольное приложение на **Rust**, представляющее собой песочницу для экспериментов с криптографическими алгоритмами и атаками на них.
Проект позволяет запускать различные алгоритмы шифрования и проверять их устойчивость к известным криптоаналитическим атакам.
На данный момент поддерживаются следующие алгоритмы:

- **RSA**
- **Diffie-Hellman**
- **El Gamal**

## Возможности
- Запуск различных атак
- Выбор сложности шифрования (размер параметров)
- Генерация **seed**, позволяющего воспроизводить эксперименты с теми же параметрами
- Сохранение результатов атак в JSON-файлы
- Возможность сравнивать эффективность различных атак

Каждый запуск атаки сохраняет отчет со следующей информацией:
- Параметры алгоритма
- Продолжительность выполнения
- Количество итераций
- Результат атаки

## Клонирование репозитория
Для клонирования репозитория выполните команду
```bash
git clone https://github.com/egor-shishkarev/sandbox-for-cryptocodes.git
cd sandbox-for-cryptocodes
```

## Требования
Для работы проекта необходимо установить **Rust**. Самый простой способ - через `rustup`:
```
https://rustup.rs/
```

## Сборка проекта

```bash
cargo build
```
Скомпилированный бинарный файл будет находиться в `target/debug`

## Запуск
Для запуска приложения выполните команду
```bash
cargo run
```
в папке репозитория

## Реализованные атаки
**RSA**:
- Brute force factorization
- Fermat factorization
- Small exponent attack

**Diffie-Hellman**:
- Brute force
- Baby-step Giant-step

**ElGamal**:
- Brute force
- Pohlig-Hellman
